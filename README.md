# Accelerometer and Giroscope_STM32

To build, pass `COMPILER_DIR` to `make`, set to path to arm-none-eabi-gcc, like this: <br  />
`$ make COMPILER_DIR=~/opt/gcc-arm-none-eabi-7-2017-q4-major/bin` <br  />
Supported `make` targets:
`build` <br />
`debug` <br />
`clean` <br />
`flash`
